# Student-Engagement-Analysis
This project analyzes student engagement data to understand patterns in course completion and learning behavior.

##  Tools Used
- SQL (Data extraction & analysis)
- Excel (Dashboard & visualization)

## Dataset
The dataset includes:
- Course category
- Time spent on course
- Videos watched
- Quiz scores
- Completion status

##  Analysis Performed
- Course Category vs Completion Rate
- Time Spent vs Completion
- Videos Watched vs Engagement
- Quiz Performance vs Completion

##  Key Insights
- Students who spend more time are more likely to complete courses
- Higher quiz scores correlate with higher completion rates
- Engagement (videos watched) significantly impacts completion

##  Dashboard
An interactive Excel dashboard was created to visualize insights and trends.

##  Project Outcome
This project helped in understanding how student behavior impacts course completion and improved my skills in SQL and data visualization
